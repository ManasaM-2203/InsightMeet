<h1>🧠 InsightMeet — AI-Powered Meeting Summarizer</h1> <h3>Missed a meeting? Don’t worry! Just upload the recording and InsightMeet will handle the rest.</h3> <p> InsightMeet is your smart assistant for transforming meeting recordings into clear, structured, and actionable insights. Upload any video or audio file, and the system will automatically transcribe, identify speakers, generate summaries, and allow intelligent Q&A using an LLM-powered chatbot. </p> <hr> <h2>🚀 Features</h2> <ul> <li>🎥 Upload meeting recordings (video/audio)</li> <li>🧠 Automatic transcription using Whisper</li> <li>🗣️ Speaker diarization (who spoke when)</li> <li>🏷️ Speaker name mapping using OCR</li> <li>🧾 Speaker-wise summaries</li> <li>📊 Full meeting summary</li> <li>💬 Chatbot for follow-up questions (LLM + Vector DB)</li> <li>📁 Structured JSON output</li> </ul> <hr> <h2>🛠️ Tech Stack</h2> <h3>Frontend</h3> <ul> <li>React.js</li> </ul> <h3>Backend</h3> <ul> <li>Python (Flask)</li> </ul> <h3>AI / ML</h3> <ul> <li>Whisper (Speech-to-Text)</li> <li>Transformers (NLP)</li> <li>Pyannote (Speaker Diarization)</li> <li>Tesseract OCR</li> </ul> <h3>Search / Retrieval</h3> <ul> <li>FAISS / ChromaDB</li> </ul> <h3>LLM (Pluggable)</h3> <ul> <li>Mistral</li> <li>LLaMA</li> <li>GPT (planned)</li> </ul> <hr> <h2>⚙️ Prerequisites</h2> <p>Make sure you have the following installed:</p> <ul> <li>Python 3.11</li> <li>Node.js & npm</li> <li>FFmpeg</li> <li>Tesseract OCR</li> </ul> <p><b>Install on Mac:</b></p> <pre> brew install ffmpeg brew install tesseract </pre> <hr> <h2>🔑 Environment Variables</h2> <p>Create a <code>.env</code> file in the root folder:</p> <pre>
